# 🚀 ssh-bot-Robexia

یک ربات قدرتمند تلگرامی SSH/SFTP که به کاربران اجازه می‌دهد مستقیماً از داخل تلگرام به سرورهای خود متصل شوند.
مناسب برای زمانی که دسترسی به اینترنت محدود است اما تلگرام همچنان در دسترس می‌باشد.

## ✨ امکانات

* ⚡ اتصال سریع SSH با استفاده از آی‌پی، نام کاربری و رمز عبور
* 🔍 تشخیص خودکار پورت پیش‌فرض `22`
* 🛠 پشتیبانی از پورت دلخواه
* 💻 ترمینال زنده داخل تلگرام
* 📤 اجرای دستورات و دریافت خروجی لحظه‌ای
* 📁 ذخیره و مدیریت لیست سرورها (**My Hosts**)
* 🔄 اتصال مجدد سریع به سرورهای ذخیره‌شده
* ⏸ خروج موقت از ترمینال با دستور `wait`
* ❌ بستن کامل ترمینال با دستور `close`
* 💤 بستن خودکار سشن در صورت عدم فعالیت
* 📂 SFTP داخلی برای مدیریت فایل‌ها
* 📤 پشتیبانی از آپلود فایل (تا 20MB)
* 🎛 پنل ساده و سریع
* ⌨ دکمه‌های میانبر هوشمند
* 📝 دکمه‌های پویا هنگام کار با nano / vim

## ⚙️ نصب سریع

این دستور را روی سرور خود اجرا کنید:

```bash id="k3m8z1"
bash <(curl -sSL https://raw.githubusercontent.com/Hajiyor/ssh-bot-Robexia/main/install.sh)
```

### نصب‌کننده به‌صورت خودکار:

* تمام پیش‌نیازها را نصب می‌کند
* توکن ربات را دریافت می‌کند
* آیدی عددی مدیر را دریافت می‌کند
* ربات را پیکربندی می‌کند
* ربات را اجرا می‌کند

---

## 🤖 دستورات ربات

| دستور       | توضیحات                           |
| ----------- | --------------------------------- |
| `/start`    | بازگشت به پنل اصلی                |
| `/fast_ssh` | اتصال سریع (بدون ذخیره)           |
| `/my_hosts` | سرورهای ذخیره‌شده (تا 5 سرور)     |
| `/close`    | بستن سشن فعلی                     |
| `/wait`     | انتقال سشن به پس‌زمینه (15 دقیقه) |

---

## 💻 داخل ترمینال

* هر متنی ارسال کنید → به‌عنوان دستور SSH اجرا می‌شود
* دکمه‌ها میانبرهای کاربردی ارائه می‌دهند
* هنگام باز بودن nano / vim دکمه‌ها به‌صورت خودکار تغییر می‌کنند

---

## 📂 حالت SFTP

* هنگام اتصال سریع، گزینه **SFTP** را انتخاب کنید
* فایل ارسال کنید → به‌صورت خودکار آپلود می‌شود
* حداکثر حجم فایل: **20MB**

---

## ⏰ زمان‌پایان سشن

سشن‌های غیرفعال پس از **5 دقیقه** به‌صورت خودکار بسته می‌شوند.
